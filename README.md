# My Tools

## Installation
1. Run with sudo.
```
sudo ./my-tools.sh
```
2. Run without sudo.
```
./oh-my-zsh.sh
```
3. Accept oh-my-zsh as default shell
4. When ZSH shells opens type "exit" and press enter so the script continues running.
